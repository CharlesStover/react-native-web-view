## Install

* `npm install rn-webview --save` or
* `yarn add rn-webview`

## Use

Instead of `import { WebView } from 'react-native'`, simply `import WebView from 'rn-webview'`.
Then continue to use `<WebView />` just like you would the React Native implementation.
